# sao-input

A Sword Art Online inspired text input and confirmation system

<br>
Text Input:
<br>

**Usage:**
- `exports["sao-input"]:TextInput("Description", 50)`


**Parameters:**
- Description: string
- Max Length: int

<br>

Confirmation:
<br>

**Usage:**
- `exports["sao-input"]:Confirm("Title", "Description")`


**Parameters:**
- Title: string
- Description: string
